## 项目核心术语

**目标类型**：SI（进货）/ SO（出货）/ ST（门店销售），月度为主。

**拆解层级（孟加拉一期固定）**：
```
国家 → 区域 → Dealer → 门店 → Promoter → 机型
```
- 客户 = Dealer（经销商）
- 人员 = Promoter（促销员）
- 一期不涉及 Supervisor 层级（已在澄清中确认）

**试点范围**：孟加拉；品牌 TECNO / Infinix。

**拆分权重（一期简单规则）**：历史达成率 / 资金盘 / 阵地数（三项权重可配置占比，和=100%）。
不含：市场容量、机型生命周期（二期）。

**自动平衡四种模式**：AI 权重 / 历史销量 / 达成率 / 平均分摊。

**强校验（≥ 模式）**：
- 区域合计 ≥ 总目标
- 对象合计 ≥ 区域目标
- 机型合计 ≥ 对象目标
- 允许超额，0 容忍不通过

**合理性指标**：统一为"目标达成率占比"，不做预测准确率、不做返利偏差率。

**Why**：两轮澄清（2026-07-02）产品经理确认的基线，与 PDF 原稿有差异时以澄清结论为准。

**How to apply**：撰写需求规格时，所有模块的术语命名保持与本表一致。
