# 目标\&达成管理MVP

### 背景与定位

1. 目标管理的核心不是把线下 Excel 搬到线上，而是构建“目标制定、目标拆解、审批生效、执行确认、达成追踪、异常处理”的闭环能力。当前一期围绕孟加拉  Infinix BP的目标管理 MVP，先解决目标如何定、如何拆、如何查看达成和如何让一线执行的问题。

2. 长期产品定位为面向多国家、多品牌、多目标对象、多目标类型的目标达成管理平台。短期聚焦孟加拉，预留全球多国家、多品牌复制能力。

### 目标用户（一期）

|角色|核心诉求|关键页面|
|---|---|---|
|零售经理|快速制定目标、查看建议、变更留痕、追踪达成|新建目标、目标详情、达成页面|
|促销员|确认目标、查看达成、接收提醒|APP 执行端|

### 目标范围（一期）

|维度|一期口径|交互影响|
|---|---|---|
|国家|孟加拉|新建目标、规则配置|
|品牌|Infinix|品牌下拉仅展示用户授权品牌；选择品牌后联动机型、规则包、审批模板。|
|目标类型|SO|一期优先落 SO，SI、ST 作为字段与页面能力预留。|
|周期|月度|新建目标默认下月；自定义周期展示开始日期和结束日期。|
|目标对象|人员|目标对象单选<br>|
|机型维度|全部、中高端机型、新品、清尾|支持快捷选择、快速查看、批量导入、批量编辑、新增和删除机型。<br>|
